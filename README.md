# Tortor ⚡️

This is a website that allows you take screenshots of other websites. Simply enter the URL and a screenshot will be taken and available for download. 

## Tools 

- Fullstack Next.js i.e. 
    - Next.js 
    - Node.js 
- chrome-aws-lambda
- Puppeteer 
- React Testing Library 
- Jest 
- TypeScript
